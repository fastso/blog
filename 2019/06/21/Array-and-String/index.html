Array and String

Introduction to Array

  • Array
    An array is a basic data structure to store a collection of elements sequentially.

  • Dynamic Array
    Most programming languages offer built-in dynamic array which is still a random access list data structure but with variable size.(ArrayList in Java)

Introduction to 2D Array

  • 2D Array
    In Java, the two-dimensional array is actually a one-dimensional array which contains M elements, each of which is an array of N integers.

  • Dynamic 2D Array
    It can be just a nested dynamic array.

Introduction to String

A string is actually an array of unicode characters.
In Java, String is immutable.
