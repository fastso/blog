GitHub Pages + Hexo 搭建个人博客

准备

使用Hexo需要用到以下环境：

  • 安装Node.js
  • Github Pages repository

安装Hexo

在 Node.js 安装好的前提下，执行 npm install 命令安装 Hexo 。

生成博客的项目。

此次生成我使用 blog 作为项目名。

移动到 blog 文件夹，安装相关的模块。

启动本地服务器。

这样就可以通过浏览器阅览 http://localhost:4000/ 博客了。

部署到GitHub

_config.yml

・・・(省略)・・・

Deployment

Docs: http://hexo.io/docs/deployment.html

deploy:
type: github
repo: git@github.com:n0bisuke/n0bisuke.github.io.git
branch: master

URL

If your site is put in a subdirectory, set url as ‘http://yoursite.com/child' and root as ‘/child/‘

url: http://fastso.github.io/blog
root: /blog/

$ hexo deploy -g

发文

$ hexo new “新規ページタイトル”

$ vi source/_posts/hexoについて.md
