[数学文章作法 推敲篇] 读书笔记

读完结城浩老师的 [数学文章作法 推敲篇] 一书，受益匪浅。
此书的内容其实不仅仅限于数学文章，
不论是日常写作还是商业邮件，都能从中得到帮助。
以下是根据书中检查表翻译后的笔记。

读者的迷惑

  • 读者通过文章构筑概念。构筑概念就像搭积木。是否给了读者正确的积木？
  • 是否给读者展示了没有分支的一条道？
  • 语句的表记是否统一？
  • 是否存在过长的句子？
  • 是否存在语言的不足？
  • 是否存在应该用图表传达的概念，却用了文章？
  • 是否存在多余的语言？
  • 指示语的所指是否明确？

推敲的基本

  • 重读的时候是否戴着读者的帽子？
  • 重读的时候手里是否拿着笔记用具？
  • 是否找到与想写的事情的差异？
  • 是否持有文章完成形象？
  • 是否删除不要的东西？
  • 是否补充必要的东西？
  • 是否将读者卷入迷雾中？
  • 是否反复读了文章？
  • 是否意识着观点再读？
  • 是否放置一段时间然后再读？
  • 是否持有作为作者的自觉性和责任感？

词语

  • 是否使用读者能够理解的词语？
  • 是否检讨过更合适的词语？
  • 是否将专业术语和一般词语区别开来？
  • 是否正确的使用专业术语？
  • 是否在有必要的情况下，记述词语的定义？
  • 是否注意定义的词性？
  • 是否欠缺考虑的创造新词？
  • 是否存在无意中使用的词语？比如说”基本上来说”，”在某种意义上”，”真正”，”绝对”，”一定”等。
  • 指示语所指示的对象是否确定？

句子的推敲

  • 是否存在过长的句子？
  • 是否存在容易被误解的复句？
  • 符号的位置和词序是否恰当？
  • 是否存在不要的句子？
  • 是否存在使用多个”的”字的句子？
  • 当存在多个概念时，对应关系是否明确？
  • 是否正确使用”各自”？
  • 是否存在容易被误解的”言外之意”？
  • 是否写有”A的情况”和”A以外的情况”？
  • 是否将复杂的概念强行写成文章？
  • 是否明确将什么与什么作比较？
  • 是否明确比较的基准？
  • 是否避免双重否定？

文章整体的平衡

  • 文章整体上，是否存在欠缺的要素？
  • 是否能一丝不漏的把读者的期望写出来？
  • 各要素的长度和强调的程度是否合适？
  • 文章是否能抵抗误读？
  • 为了避免误读，是否加入了适当的冗余性？
  • 是否存在粗糙的部分混在其中？
  • 是否整体的读过？（车轮战）
  • 是否改变观点读过好多次？（阶段战）
  • 是否仔细阅读修改较多的地方？（张弛战）

审查

  • 是否考虑让他人审查？
  • 是否考虑审查者是谁？（对象读者，专家，对文章敏感的人等）
  • 是否给了审查者审查项目？
  • 是否传达了审查观点？
  • 是否考虑使用审查专用网页？
  • 是否认真确认回馈信息？
  • 是否将回馈信息分类，整理？
  • 是否冷静的接受回馈信息？
  • 是否以作者的责任感来对应回馈信息？
  • 是否把握没有对应的回馈信息？
  • 是否和审查者保持着信赖关系？

推敲的窍门

  • 是否尽力保守期限？
  • 是否把握到期限为止的总时间？
  • 是否把握有无整段时间可用？
  • 为了有效的推敲，是否制作词汇表？
  • 是否使用检索，替换，和校准功能？
  • 是否掌握自己写文章时的习惯？
  • 是否对文件进行备份？
  • 是否使用版本管理系统？
  • 是否使用多种读法阅读？（出声读，画面读，纸面读，换地方读，随机的读某页等）
