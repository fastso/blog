5. Underfitting and Overfitting

Experimenting With Different Models

  • Overfitting
    A model matches the training data almost perfectly, but does poorly in validation and other new data.
    (capturing spurious patterns that won’t recur in the future, leading to less accurate predictions)

  • Underfitting
    A model fails to capture important distinctions and patterns in the data, so it performs poorly even in training data.
    (failing to capture relevant patterns, again leading to less accurate predictions.)

Why Do People Use Python?

  • Software quality
    The uniformity of Python code makes it easy to understand.

  • Developer productivity
    Python code is typically one-third to one-fifth the size of equivalent C++ or Java code.That means there is less to type, less to debug, and less to maintain.

  • Program portability
    Most Python programs run unchanged on all major computer platforms.

  • Support libraries

  • Component integration
    Python code can invoke C and C++ libraries, can be called from C and C++ programs, can integrate with Java and .NET components.

  • Enjoyment
    Because of Python’s ease of use and built-in toolset, it can make the act of programming more pleasure than chore.

Is Python a “Scripting Language”?

general-purpose programming language that blends procedural, functional, and object-oriented paradigms

  • Shell tools
    coding operating-system-oriented scripts

  • Control language
    control and direct other application components.

Execution speed is the only major downside to Python.

The java.util.function Package

The java.util.function package was added in Java 8.
The interfaces in java.util.function fall into four categories.

  • consumers : take a generic argument and return nothing
  • suppliers : take no arguments and return a value
  • predicates : take an argument and return a boolean
  • functions : take a single argument and return a value

GitHub Pagesに独自ドメインを設定

GitHub Pagesに独自ドメインを設定する手順は、ほかのサイトで紹介されているのでここでは述べない。
