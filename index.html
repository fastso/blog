Default Methods in Interfaces

Provide an implementation of a method inside an interface.

An Employee interface with a default method

Static Methods in Interfaces

Provide a class-level utility method to an interface, along with an implementation.
Static methods in interfaces remove the need to create separate utility classes,
though that option is still available if a design calls for it.

key points

  • Static methods must have an implementation
  • Cannot override a static method
  • Call static methods from the interface name
  • Do not need to implement an interface to use its static methods
